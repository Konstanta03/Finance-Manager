##  Finance Manager

###  Опис проєкту

**Finance Manager** — це десктопна аплікація для обліку особистих фінансів, розроблена з метою зручного контролю доходів і витрат. Вона допомагає користувачам аналізувати фінансові потоки, відстежувати баланс та формувати звіти для прийняття обґрунтованих фінансових рішень.

---

##  Мета застосунку

* Надати зручний інструмент для ведення особистого або сімейного бюджету.
* Допомогти користувачам визначити основні статті витрат.
* Формувати статистику для оптимізації фінансових витрат.
* Забезпечити наочну візуалізацію фінансових даних.

---

##  Типи користувачів

* **Звичайний користувач** — веде власний бюджет та аналізує персональні витрати.
* **Сім’я / група** — декілька користувачів можуть вести спільний облік фінансів.

---

---

##  Основний функціонал

* Додавання, редагування та видалення транзакцій (доходи та витрати).
* Категоризація витрат (їжа, транспорт, розваги, інше).
* Пошук і фільтрація операцій за датою та категорією.
* Автоматичний підрахунок поточного балансу.
* Формування статистики:

  * щомісячні витрати,
  * топ-категорії,
  * графіки та діаграми.

---

##  Аналіз подібних застосунків

### **Wallet (BudgetBakers)**

 Зручний інтерфейс
 Синхронізація з банківськими рахунками
 Повна версія платна, існують обмеження у безкоштовній

### **Spendee**

 Сучасна та приваблива візуалізація витрат
 Підтримка сімейних акаунтів
 Обмежений функціонал без підписки

### **Money Lover**

 Проста система категорій
 Нагадування про регулярні платежі
 Надокучлива реклама у безкоштовній версії
